# MiniiERP1

A REST API for managing inventory and order management system.

## Project Overview

This project is designed to provide a comprehensive solution for inventory tracking and order management within a mini-ERP system. It aims to streamline business operations by offering functionalities for product management, stock control, and order processing.

## Features

- Product catalog management
- Inventory tracking
- Order processing and management
- User authentication and authorization
- RESTful API endpoints

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

- .NET SDK
- Visual Studio or VS Code
- SQL Server or SQLite

### Installation

1. Clone the repository
2. Restore NuGet packages
3. Build the solution
4. Run the application
